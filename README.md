# EcoBuddy 🌍

This is the codebase for **EcoBuddy**, a modern sustainability tracker app designed to help you monitor and improve your environmental impact.

The original project design is available on [Figma](https://www.figma.com/design/91W6q52l3HugCztqjjjllt/ECOHUB-Sustainability-Tracker).

## 🚀 Live Demo

The application is deployed and live at: **[https://ecobuddy-app.netlify.app](https://ecobuddy-app.netlify.app)**

## 💻 Running the code locally

To get this project up and running on your local machine, follow these steps:

1. **Install the dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
